Thank you for contributing to Recipe_Book!

How to contribute
- Fork the repo and create a branch named fix/feature-your-name.
- Add or update recipes in the recipes/ directory as Markdown files.
- Follow the recipe template in templates/RECIPE_TEMPLATE.md.
- Open a pull request with a short description of your changes.

Recipe formatting guidelines
- Use the provided template.
- Keep ingredient lists one item per line.
- Use numbered steps for instructions.
- Add tags (cuisine, diet, time) in the front-matter section.
- Include a photo in assets/ and reference it by path.

Images
- Prefer JPEG/PNG, keep files under 2MB when possible.
- Place images in assets/ and use descriptive filenames (e.g., pad-thai-plate.jpg).

License and attribution
- By submitting a recipe you agree to license it under the repository license (see LICENSE).
- If the recipe was adapted from another source, add a short attribution line.

Code of conduct
- Be respectful and constructive in PR comments and reviews.
